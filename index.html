Steps to set up:

  1. Clone the your bot:
    2. git clone https://github.com/Tamino1230/customer-discord-bot.git
  2. Navigate to the project directory: cd discord-bot
  3. Install dependencies:
    4. pip install -r requirements.txt
  4. Set up your environment variables in the .env file.
  5. Set up your bot in the Discord Developer Portal and get your bot token.
  6. Configure the bot by editing the config.py file.
  7. Run the bot: (can depend)
    8. python main.py
  8. Invite the bot to your server using the OAuth2 URL generated in the Discord Developer Portal with Admin permissions.
